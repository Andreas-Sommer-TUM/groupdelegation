===================
How to contribute
===================

Contribution rules
=====================

- There must be a ticket in the project's `bug tracker`_ explaining the problem / the suggested enhancement
- `PSR-2`_ coding guidelines are enforced
- Commit message complies to the `format used by TYPO3`_ (the "releases:" line is useless here)
- Unique logical change per patch
- The commit message must refer to the ticket number on github like `Resolves #9`

.. _bug tracker: https://github.com/in2code-de/groupdelegation/issues
.. _PSR-2: http://www.php-fig.org/psr/psr-2/
.. _format used by TYPO3: https://wiki.typo3.org/CommitMessage_Format_(Git)
